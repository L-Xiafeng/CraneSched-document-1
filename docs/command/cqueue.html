4.3 cqueue 查看作业队列

cqueue 查看作业队列

cqueue可以查看队列中的作业信息。

查看集群中所有队列的作业信息。

cqueue

cqueue运行结果展示

主要输出项

  • TaskId：作业名
  • Type： 作业类型
  • Status：作业状态
  • NodeIndex： 作业运行的节点名

主要参数

  • --help: 显示帮助
