4 Crane使用教程

Crane使用教程

Crane有以下6个常用指令，请点击查看更详细使用信息

  • cinfo： 查看节点与分区状态
  • cbatch： 提交批量处理作业
  • cqueue： 查看作业队列
  • ccancel： 取消作业
  • ccontrol:查看分区和节点状态
  • cacctmgr： 查看和调整用户/账号信息
  • cacct： 查看作业信息
